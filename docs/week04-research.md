# BÁO CÁO TUẦN 4

## 1. Semantic HTML

Semantic HTML là việc sử dụng các thẻ HTML có ý nghĩa ngữ nghĩa (semantic meaning) để mô tả đúng vai trò và nội dung của phần tử trong tài liệu.

→ Mục đích: giúp trình duyệt, công cụ tìm kiếm và công nghệ hỗ trợ hiểu cấu trúc nội dung.

## 2. CSS Selectors

CSS Selectors là các biểu thức được sử dụng để xác định và chọn các phần tử HTML mà quy tắc CSS sẽ được áp dụng.

→ Chúng định nghĩa phạm vi áp dụng style trong tài liệu.

## 3. Box Model

Box Model là mô hình biểu diễn mỗi phần tử HTML như một hộp hình chữ nhật gồm 4 lớp:

- Content
- Padding
- Border
- Margin

→ Nó xác định kích thước và khoảng cách hiển thị của phần tử trên trang.

## 4. BEM Methodology

BEM (Block – Element – Modifier) là một phương pháp đặt tên CSS nhằm tổ chức và cấu trúc code theo thành phần độc lập.

- Block: thành phần độc lập
- Element: phần tử con của block
- Modifier: biến thể của block/element

→ Mục tiêu: tăng tính tái sử dụng và dễ bảo trì.

## 5. Viewport

Viewport là vùng hiển thị nội dung của trang web trên thiết bị người dùng.

→ Nó xác định kích thước và tỷ lệ hiển thị của layout, đặc biệt quan trọng trên thiết bị di động.

## 6. Media Queries

Media Queries là kỹ thuật CSS cho phép áp dụng style dựa trên đặc điểm của thiết bị hoặc viewport (ví dụ: chiều rộng, độ phân giải).

→ Dùng để xây dựng responsive design.

## 7. Mobile-first Design

Mobile-first design là phương pháp thiết kế giao diện bắt đầu từ thiết bị di động trước, sau đó mở rộng lên các màn hình lớn hơn.

→ Nguyên tắc: ưu tiên nội dung cốt lõi và hiệu năng.

## 8. Đơn vị tương đối (rem, em, vh, vw)

- rem (root em): Đơn vị chiều dài tương đối dựa trên font-size của phần tử gốc (html).

- em: Đơn vị tương đối dựa trên font-size của phần tử cha gần nhất.

- vw (viewport width): Đơn vị bằng 1% chiều rộng của viewport.

- vh (viewport height): Đơn vị bằng 1% chiều cao của viewport.
